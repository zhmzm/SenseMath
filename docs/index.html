SenseMath

Evaluating Number Sense in Large Language Models
Haomin Zhuang, Xiangqi Wang, Yili Shen, Ying Cheng, Xiangliang Zhang
University of Notre Dame
📄 Paper 💻 Code 📊 Dataset

Abstract

Large language models default to step-by-step computation even when efficient numerical shortcuts exist. We introduce SenseMath, a controlled benchmark that measures whether LLMs can exploit number-sense shortcuts when prompted. The benchmark comprises 1,600 matched item families spanning 8 categories and 4 digit scales, each with strong-shortcut, weak-shortcut, and control variants. We evaluate 5 models under chain-of-thought (CoT), number-sense (NS), and strict prompting conditions. We find that NS prompting induces a broad shortcut-seeking mode: accuracy on shortcut-amenable items is maintained or improved while control accuracy drops. Per-category analysis reveals that models show selective strengths and reversals across shortcut types, establishing number sense as a multi-dimensional construct. A three-level evaluation framework (Use, Judge, Generate) confirms that shortcut exploitation, recognition, and construction dissociate across models.

Main Results

NS prompting effect across digit scales and models. NS maintains strong-item accuracy while reducing control accuracy.
Key mechanism: NS prompting uniformly increases shortcut strategy usage (from 20–39% under CoT to 68–86% under NS), but these strategies succeed on strong items (where valid shortcuts exist) and fail on control items (where no shortcut applies), producing the observed accuracy asymmetry.
